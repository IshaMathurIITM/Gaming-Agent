# Gaming-Agent
This repository features a Reinforcement Learning (RL) agent built to master the classic CartPole-v1 environment.  

Powered by PyTorch and Gymnasium, the agent utilizes a Deep Q-Network (DQN) to autonomously learn the optimal strategy for balancing a pole on a moving cart. Through trial, error, and neural network-based decision-making, the agent transitions from taking random actions to executing flawless balance control.
